MySQL Connector/Net

Installation and Deployment

MySQL Connector/Net is provided in two different packages. One is a zip file that includes the assemblies and documentation files. The other package includes an MSI-based installer. This installer will allow you to install the software to any location and give you the option of registering the assemblies into the Global Assembly Cache (GAC). You can also choose to register the product so that it appears in the Visual Studio .NET Add Reference dialog.

Other products require different steps to perform the installation. The following topics include more detailed installation instructions.

  • Installing into Visual Studio.Net
  • Installing into Borland Delphi and C# Builder
  • Installing into Microsoft Web Matrix
