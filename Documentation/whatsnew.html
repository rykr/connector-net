MySQL Connector/Net

What's New

Connector/Net 1.0.4 new features:

  • Support for all MySQL versions 3.23 and higher
  • TCP, Unix socket, named pipe, and shared memory connections are supported
  • Works with Net 1.0, 1.1, and Mono 1.0
  • Support for compressed connections
  • Supports transactions on transactional enabled storage engines
  • Supports stored procedures on MySQL 5.0.x
